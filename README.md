# Keele Campus Snake Game

## Project Overview
This repository contains a simple, interactive Snake game built using Python's `tkinter` library. This project was developed as part of **Assessment 2** for the module **Foundations of Programming and Software Engineering (CSC-44102)** at Keele University.

The primary objective of this assessment is to demonstrate a professional software engineering workflow using:
*   **Git** for local version control.
*   **GitHub** for remote repository management.
*   **Branching and Merging** strategies for feature development.
*   **Generative AI Co-creation** as permitted by the assessment specification.

## Features
*   Classic Snake gameplay with directional controls.
*   Real-time scoring system.
*   Collision detection (Walls and Self).
*   Graphical User Interface (GUI) built with Tkinter.

## How to Run
1.  Ensure you have Python 3.x installed on your system.
2.  Clone this repository:
    ```bash
    git clone https://github.com/zion-umoh/Keele-Snake-Game.git
    ```
3.  Navigate to the project directory:
    ```bash
    cd Keele-Snake-Game
    ```
4.  Run the game:
    ```bash
    python snake_game.py
    ```

## Controls
*   **Arrow Keys (Up, Down, Left, Right):** Change the snake's direction.
